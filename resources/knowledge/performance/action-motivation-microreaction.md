# Action Motivation And Microreaction

更新时间：2026-05-22

用途：让角色有“活人感”，把表演从站桩、微笑、看镜头，改成有处境、有动机、有小反应的动作。

## 核心判断

人物真实感不是更精致的脸，而是“她为什么在这一刻这么动”。

每个角色动作都要写：

- 当前处境：角色现在处于什么压力或期待里。
- 动作动机：是什么触发了动作。
- 身体反应：呼吸、肩膀、手指、重心、嘴角、眼皮如何轻微变化。
- 视线方向：看人、看物、看画外、避开视线。

## 单人表演公式

```text
身份 + 处境 + 触发 + 主动作 + 微反应 + 镜头观察方式
```

示例：

```text
短发少女站在雨后屋顶，手里握着旧磁带。
她听到少年脚步声后没有马上抬头，先轻轻收紧手指。
半秒后她抬眼看向画面右侧，呼吸让肩线轻微起伏。
镜头中近景慢慢推进，保留她犹豫的停顿。
```

## 多人表演公式

多人镜头里只让一个人主动作，另一个人反应。

```text
A 行动：递出 / 后退 / 伸手 / 转身。
B 反应：停顿 / 看向道具 / 皱眉 / 接住 / 避开。
```

不要写两个人同时“深情对视并走近又递东西”。这类描述会让动作平均化。

## 微反应库

- 眨眼慢半拍。
- 指尖收紧后松开。
- 视线先看道具，再看对方。
- 肩膀随呼吸轻微起伏。
- 嘴角想说话但停住。
- 重心从后脚移到前脚。
- 接住道具后手腕微微下沉。

## QA 检查

- 角色动作是否有触发原因。
- 微反应是否小而具体。
- 角色是否没有无意义地看镜头。
- 多人镜头是否避免同时复杂表演。
